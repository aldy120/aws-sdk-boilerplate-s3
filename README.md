A boilerplate for [AWS SDK JavaScript v3](https://docs.aws.amazon.com/AWSJavaScriptSDK/v3/latest/index.html)

# Install yarn
```
npm install -g yarn
```

# Git clone
```
git clone https://github.com/aldy120/aws-sdk-boilerplate-s3.git
cd aws-sdk-boilerplate-s3
```

# Install packages
```
yarn install
```

# DynamoDB example
```
node dynamodb.js
```
